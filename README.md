# React + TypeScript + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.
## Prerequisites

Make sure you have the following installed before running the project:

### 1. Node.js & npm
- Download and install from [nodejs.org](https://nodejs.org/)
- Recommended version: **v18 or higher**
- npm comes bundled with Node.js — no separate install needed
- Verify installation:
```bash
  node --version
  npm --version
```

### 2. Git
- Download and install from [git-scm.com](https://git-scm.com/)
- Used to clone the repository
- Verify installation:
```bash
  git --version
```

### 3. A Code Editor (optional but recommended)
- [Visual Studio Code](https://code.visualstudio.com/) is recommended

---

## Getting Started

Clone the repository:
```bash
git clone https://github.com/your-username/spectra.git
cd spectra
```

Install all dependencies including Electron:
```bash
npm install
```

Run the app:
```bash
npm run dev           # runs in browser at localhost:5173
npm run electron:dev  # runs as desktop app
```

> Electron does not need to be installed separately — it is included as a project dependency and installed automatically via `npm install`.
